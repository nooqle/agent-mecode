# Agent MeCode 视觉设计

版本: 1.0.0

## 设计理念

Agent MeCode 身份卡片设计目标：

1. **人类可读**: 清晰的视觉层次结构
2. **机器可读**: 为 AI Agent 嵌入数据
3. **独特风格**: 独特的赛博朋克/像素艺术美学
4. **紧凑布局**: 卡片格式包含所有必要信息

## 视觉效果

### 发光效果
主要元素有微妙的发光：
```css
text-shadow: 0 0 10px currentColor;
filter: drop-shadow(0 0 5px currentColor);
```

### 边框发光
卡片边框有动画发光：
```css
box-shadow: 0 0 20px rgba(0, 255, 213, 0.3);
```

### 像素艺术元素
- 角落装饰性像素图案
- 复古风格分隔线
- 8位风格图标

## 无障碍设计

- 最小对比度: 4.5:1
- 所有文字可选择
- 视觉元素有替代文本
- 元数据中有机器可读数据

## 动画 (可选)

当 `animated: true` 时：
- 微妙的边框脉冲 (2秒周期)
- 发光强度变化
- 无干扰性动画

## 导出格式

| 格式 | 用途 |

卡片缩放时保持宽高比：
- 最小宽度: 200px
- 最大宽度: 600px
- 文字按比例缩放
